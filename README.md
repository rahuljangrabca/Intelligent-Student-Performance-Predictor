# Intelligent Student Performance Predictor

A comprehensive MERN stack application designed to track, predict, and improve student academic performance using data analytics.

## 🚀 Features

### For Teachers
- **Dashboard Overview**: Real-time stats on students, courses, and active classes.
- **Attendance Management**: Digital attendance register with automatic performance updates.
- **Performance Predictor**: AI-driven risk analysis (Low, Medium, High) based on attendance and scores.
- **Class Scheduling**: Manage daily classes, virtual sessions (Live), and rescheduling requests.
- **Student Analytics**: Detailed breakdown of student strengths, weaknesses, and suggestions for improvement.

### For Students
- **Personal Dashboard**: Track attendance, GPA, and upcoming deadlines.
- **Course Enrollment**: Browse and enroll in available classes.
- **Task Management**: Keep track of assignments and personal tasks.
- **Real-time Chat**: Interaction with teachers and peers.

## 🛠️ Technology Stack
- **Frontend**: React.js, Vite, Lucide React, Glassmorphism CSS.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).
- **Authentication**: JWT & Bcrypt.

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rahuljangrabca/Intelligent-Student-Performance-Predictor.git
   ```

2. **Backend Setup**:
   ```bash
   cd server
   npm install
   # Create a .env file with MONGO_URI and JWT_SECRET
   npm start
   ```

3. **Frontend Setup**:
   ```bash
   cd client/FRONTEND
   npm install
   npm run dev
   ```
